[red]#注意事项#

. 使用环境为Linux
. Linux需要安装go环境
. go需要配置环境
[source]
----
  go env -w GO111MODULE=on
  go env -w GOPROXY=https://goproxy.cn,direct
  go get github.com/vishvananda/netlink
----
